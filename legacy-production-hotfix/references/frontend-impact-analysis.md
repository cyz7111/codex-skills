# 前端影响面分析

## 影响范围维度

分析潜在改动会影响哪些对象：

- 页面：路由、页面入口、子组件、弹窗、列表、表单、详情页。
- 数据：接口、请求参数、响应字段、缓存、store、local/session storage。
- 配置：环境配置、业务配置、feature flag、渠道配置、平台配置。
- 公共能力：请求封装、鉴权、错误处理、loading/toast、埋点、权限。
- 平台：Web、H5、微信小程序、支付宝小程序、抖音小程序、UniApp、微应用、华为元服务。
- 构建发布：构建命令、环境变量、包体积、分包、发布脚本。

## 高风险改动

以下改动必须列受影响调用方：

- 公共 request client、baseURL、鉴权、拦截器。
- 公共组件、公共 hooks、公共 utils。
- 全局 store、路由守卫、权限判断。
- 环境配置、构建配置、平台配置。
- 支付、登录、订单、停车、充电、券、追缴等关键链路。
